الخطوة 1 — إنشاء Google Sheet

  1. اذهب إلى sheets.google.com وأنشئ جدولاً جديداً
  2. سمّه: طلبات تركيب عدادات الماء
  3. أضف هذه الرؤوس في الصف الأول:
    رقم الطلب | التاريخ والوقت | الاسم | البطاقة الوطنية | الهاتف | رقم الاشتراك | رقم العقار | الموقع | الملاحظات | اللغة

الخطوة 2 — إنشاء Google Apps Script

  1. في Google Sheet، اختر: الامتدادات ← Apps Script
  2. احذف الكود الموجود والصق الكود التالي:
const SHEET_NAME = "Sheet1";

function doPost(e) {
  const sheet = SpreadsheetApp
    .getActiveSpreadsheet()
    .getSheetByName(SHEET_NAME);

  const data = JSON.parse(e.postData.contents);

  sheet.appendRow([
    data.ref,
    data.timestamp,
    data.name,
    data.nid,
    data.phone,
    data.subscription,
    data.property,
    data.location,
    data.notes,
    data.lang
  ]);

  return ContentService
    .createTextOutput(JSON.stringify({ status: "ok", ref: data.ref }))
    .setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  return ContentService
    .createTextOutput("Water Meter Request API is running.")
    .setMimeType(ContentService.MimeType.TEXT);
}
  1. احفظ السكريبت (Ctrl+S)

الخطوة 3 — نشر السكريبت

  1. اضغط Deploy ← New deployment
  2. النوع: اختر Web app
  3. في Execute as: اختر Me
  4. في Who has access: اختر Anyone
  5. اضغط Deploy ثم وافق على الصلاحيات
  6. انسخ رابط Web App URL الذي يظهر — يبدأ بـ https://script.google.com/macros/s/...

الخطوة 4 — ربط النموذج بالسكريبت

  1. افتح ملف water_meter_form.html بأي محرر نصوص
  2. ابحث عن السطر: const SCRIPT_URL = 'https://script.google.com/macros/s/AKfycbwajaKH8iEmmMB0KEA0h3m5TsMOKQSsK_6mnbgBBIn0cDwri6gfSHpIFUEsDcEhD7XyNw/exec';
  3. استبدل https://script.google.com/macros/s/AKfycbwajaKH8iEmmMB0KEA0h3m5TsMOKQSsK_6mnbgBBIn0cDwri6gfSHpIFUEsDcEhD7XyNw/exec برابط Web App الذي نسخته
  4. احفظ الملف

الخطوة 5 — نشر الصفحة مجاناً (GitHub Pages)

  1. أنشئ حساباً في github.com (مجاني)
  2. أنشئ Repository جديد، اسمه مثلاً: water-meter-kirkuk
  3. ارفع ملف water_meter_form.html واسمه index.html
  4. اذهب إلى Settings ← Pages ← Source: main branch
  5. ستحصل على رابط مثل: https://yourusername.github.io/water-meter-kirkuk

الخطوة 6 — توليد QR Code

  1. اذهب إلى qr.io أو qrcode-monkey.com
  2. الصق رابط الصفحة
  3. حمّل QR Code بصيغة PNG بدقة عالية
  4. اطبعه وضعه في مكتب الاستقبال أو على اللافتة الخارجية
